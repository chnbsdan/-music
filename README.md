# 🎯 使用指南
## 搜索音乐
在搜索框输入歌曲名、歌手名或专辑名

点击"搜索"按钮或按回车键

从搜索结果中选择歌曲播放

## 播放控制
播放/暂停: 点击播放按钮

上一首/下一首: 使用导航按钮

## 进度控制: 拖动进度条

音量调节: 拖动音量条

播放模式: 循环、单曲、随机

## 收藏功能
点击歌曲旁的收藏按钮添加到收藏

在"我的收藏"标签页查看收藏歌曲

支持离线访问收藏内容

## 下载歌曲
点击歌曲旁的下载按钮

或使用右键菜单下载

自动下载为MP3格式

# 🔧 自定义配置
## 添加新的API
在代码中找到 API_BASES 数组，添加新的API地址：
```
const API_BASES = [
    // ... 现有API
    'https://your-new-api.com'
];
```
## 修改音质设置
在 qualityMap 对象中调整音质映射：
```
const qualityMap = {
    standard: 'standard',
    high: 'higher', 
    lossless: 'exhigh'
};
```

## 请求格式
搜索: /search?keywords=关键词&limit=数量

歌曲详情: /song/detail?ids=歌曲ID

音频URL: /song/url?id=歌曲ID

歌词: /lyric?id=歌曲ID

# 🐛 常见问题
Q: 搜索不到歌曲怎么办？
A: 尝试更换搜索关键词，或使用随机推荐功能

Q: 播放卡顿怎么办？
A: 切换到标准音质，或检查网络连接

Q: 下载失败怎么办？
A: 可能是网络问题，请稍后重试

Q: 如何清除收藏？
A: 在浏览器中清除本地存储数据
